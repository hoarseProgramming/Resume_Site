Here is a brief summary of some of the projects I've been working on.

Hoarse Dungeon Crawler

This was the first "real" application I made. A game run in the console where the player navigates a dungeon, exploring more and more of their surroundings with every step, and trying their might against dangerous foes.

In this project I learned a lot about object oriented programming and how to structure a project with multiple classes and methods. I also learned about the importance of planning and how to work with a project over a longer period of time.

This game has been a work in progress for a long time and I've been coming back to it every now and then. Most recently for a bigger update where I added ascending and descending different levels but most important, the ability to save your game in a MongoDB database for loading at a later time.

Git: Hoarse Dungeon Crawler

Bookstore app

This is my first dabbling with a relationship database connected with an application. The app lets you as a user take control of the inventory balances of your very own book store.

You can add and remove books from the inventory from your stores. You can also add and remove books from your "catalog" which is a list of books that you've chosen to be able to sell in your stores.

In this project I learned a lot about communicating with a SQL server database, working with offline lists of books and figuring out how to get the entity framework change tracker to work the way I wanted it to. The user experience may not be great at times, but it does it's job.

Git: Hoarse bookstore app
